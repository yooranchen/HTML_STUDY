西施越溪女，出自苎萝山。
　　秀色掩今古，荷花羞玉颜。
　　浣纱弄碧水，自与清波闲。
　　皓齿信难开，沉吟碧云间。
　　勾践徵绝艳，扬蛾入吴关。
　　提携馆娃宫，杳渺讵可攀。
　　一破夫差国，千秋竟不还。
